# Strings Preparation Guide

String problems test logic, edge-case handling, and implementation speed.

## What to Learn

- Frequency map techniques
- Two pointers in strings
- Pattern matching basics
- Palindrome and substring strategies

## Practice Routine

1. Solve 3 easy problems daily
2. Solve 2 medium problems every alternate day
3. Revisit mistakes at the end of each week

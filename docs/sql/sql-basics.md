# SQL Basics for Placements

SQL is frequently tested in aptitude rounds, coding rounds, and interviews.

## Core Topics

- `SELECT`, `WHERE`, `ORDER BY`, `GROUP BY`
- Joins (`INNER`, `LEFT`, `RIGHT`)
- Aggregate functions and `HAVING`
- Subqueries and CTE basics

## Practice Plan

1. Learn syntax from beginner tutorials
2. Solve query sets daily on online judges
3. Practice company-level SQL interview questions
